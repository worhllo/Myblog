「CF项目」稳定版脚本与优选IP工具

Note

用 Cloudflare Worker 搭建的节点项目，没有花里胡哨的功能，只保留了最核心的上网功能，为你提供一个稳定、简洁的 Worker 脚本，并搭配一个强大的优选 IP 工具，该版本经过了大量精简和代码混淆，只保留了最核心的 VLESS 代理功能，并内置了一个非常智能的故障转移机制。

脚本特点：

  • 极致精简：移除了所有不必要，大幅提升稳定性。
  • 代码混淆：降低被识别的风险。

  1. 获取代码：
    前往我的 GitHub 项目地址获取最新的 少年你相信光吗 代码。

    https://github.com/byJoey/cfnew

  2. 创建 Worker：
    登录 Cloudflare Dashboard，进入 Workers 和 Pages > 创建应用程序 > 创建 Worker。

  3. 部署脚本：
    将 少年你相信光吗 的代码完整粘贴到编辑器中，点击 “保存并部署”。

  4. 配置环境变量（重要！）：
    部署后，进入 Worker 的 设置 > 变量，添加以下环境变量：

    • u：你的 UUID，这是必要的身份认证。
    • p：你的 ProxyIP ，可选但强烈推荐。可以填写一个稳定的域名，。

“关于 130 个节点只有一条能用这件事情，我也表示很尴尬。”

相信很多人都遇到过这个问题：网上找的优选 IP 大部分都无法使用。为了解决这个问题，我制作了一个配套的 HTML 工具，它可以帮你一键生成所有可用的节点。

工具特点：

  • 自动获取最新 IP：工具会自动从可靠的 IP 发布页抓取最新的 IPv4 和 IPv6 优选 IP 列表。
  • 生成全部可用端口：为每一个获取到的优选 IP，自动生成所有 Cloudflare 支持的 TLS 端口（443, 8443, 2053, 2083, 2087, 2096）的 VLESS 链接。
  • 一键复制：生成结果后，可以一键复制所有链接，直接导入客户端。
  • 纯前端，无后端：下载后在本地浏览器打开即可使用，安全放心。

  1. 打开网址：
    https://cfy.jhb.ovh/

  2. 填写信息：

    • Worker 项目地址：填写你刚才部署的 Worker 的域名（例如 your-worker.your-account.workers.dev）。
    • UUID：填写你在环境变量中设置的 UUID。

  3. 生成并导入：
    点击 “生成链接”，稍等片刻，下方文本框就会出现大量的 VLESS 链接。点击 “复制全部” 按钮，然后打开你的客户端（如，v2rayN）进行导入，即可开始测速和使用。

  • 关于测速：友情提醒，最好不要用 Speedtest 之类的网站做宽带测速。因为我们用的是 ProxyIP，如果大家都去跑测速，会造成 ProxyIP 失效。如果你非要测，可以用 google 的 测速网站 https://gfiber.com/speedtest/。
  • 关于速度：我的延迟不代表你的延迟，我的速度也不代表你的速度。这个方案只能保证你能用，但我无法保证你的速度跟我的速度一样，请不要抱太高的期待。
